---
Use bullet points (`- `) instead of numbered lists in `.agent/SESSION_LEARNINGS.md`.

**Why:** Numbered lists require tracking the last number in each section. Inserting new entries mid-section breaks sequential numbering. Bullets are order-independent and easier to maintain.

**How to apply:** When adding new rules to SESSION_LEARNINGS.md, use `- ` prefix. Don't number them.
